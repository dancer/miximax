# MixiMax

A companion app for **Inazuma Eleven: Victory Road** - browse players, compare stats, and build your dream team.

## Features

- **4,840+ Players** with full stats, elements, positions, and playstyles
- **422 Equipment** items with stat bonuses
- **597 Hissatsu** techniques with power and TP costs
- **48 Tactics** with effects and cooldowns
- **133 Hyper Moves** (awakenings, keshins, totems)
- **Team Builder** with 12 formations, share, export, import
- **Match Drops** reference for passive skills
- **Multi-language** (English, Japanese, German, French)
- **Light/Dark Mode**
- **JP Names** toggle for player names
